# Mundial 2026 · Mobile Horizontal Zoom

Versión para **celular en pantalla horizontal**, con **zoom y desplazamiento**.

## Qué hace

- App web móvil / PWA.
- Diseñada para usar el celular en horizontal.
- Muestra el cuadro completo tipo infograma.
- Permite zoom con controles:
  - 50%
  - 75%
  - 100%
  - 125%
  - 150%
- También permite pinch-to-zoom del navegador.
- Botón para actualizar datos reales desde FIFA.
- Final y tercer puesto incluidos.
- Banderas junto a selecciones.
- Sin pandas.
- Sin Streamlit.

## Uso ideal

Publicarla online en Render, Railway, PythonAnywhere o Replit.

## Render

1. Subir carpeta a GitHub.
2. Render → New → Web Service.
3. Conectar repo.
4. Deploy.
5. Abrir URL desde el celular.
6. Agregar a pantalla de inicio.

## En celular

- Girá el celular en horizontal.
- Usá los botones de zoom.
- Desplazate con el dedo por el cuadro.
